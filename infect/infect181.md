# Question
Which antibiotic binds directly to the D-ala-D-ala sequence of bacterial cell walls?   **Vancomycin**

---

# Answer
whereas Î²-lactams mimic this sequence to bind PBPs, vancomycin binds directly to this sequence inhibiting cell wall synthesis   this allows vancomycin to be used even when PBPs are altered (e.g. MRSA)
